# Problem

주어진 숫자부터 0까지 순서대로 찍어보세요

아래는 입력된 숫자가 N일 때 거꾸로 출력하는 예시입니다

![git1](https://user-images.githubusercontent.com/79297596/126459112-8ce4c484-b9c8-4b79-b4b0-8ab6dc48c771.PNG)

---------------------
# Solution
```
t = int(input())
for i in range(t, -1, -1):
    print(i, end=' ')
```
