# Problem

1부터 주어진 횟수까지 2를 곱한 값(들)을 출력하시오.

주어질 숫자는 30을 넘지 않는다.

-----------------------

# Solution

```
t = int(input())
db = 1
for i in range(t+1):
    print(db, end=' ')
    db = db*2
```
